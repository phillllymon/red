GRAFFITI is a browser extension that lets you view and leave comments on any webpage. It is much better looking than this website.

GRAFFITI is in the Chrome Web Store!

There may be newer features in the latest beta, which you can download here.
Note: By downloading GRAFFITI, you agree to our terms and privacy policy.

Steps to install:
1. Download the zip file linked above
2. Unzip the file. You should get a folder called "graffiti"
3. Open Chrome and go to "chrome://extensions/"
4. In the upper-right corner toggle on "Developer mode"
5. Click the button "Load unpacked"
6. Select the unzipped folder. This should install the extension
7. Click the puzzle piece icon in the upper right of Chrome
8. In the dropdown menu, click the pin next to Graffiti
Now you should see the little speech bubble icon in the browser. Click it to open GRAFFITI. It should be gray on pages that don't have comments and orange on pages that do.
